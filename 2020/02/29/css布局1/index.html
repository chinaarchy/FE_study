# 居中布局

# 水平居中

  1. inline-block + text-align
    • 父元素设置居中，自元素变成行内块
    • 具有较好的兼容性
  2. table + margin
    • 如果子级元素脱离文档流，导致margin无效
  3. absolute + transform
    • transform可能存在兼容性问题

# 垂直居中

  1. table-cell + vertical-align
  2. absolute + transform

# 居中布局

  1. table + margin + table-cell + vertical-align
  2. absolute + transform

# 多列布局

# 两列布局

  1. float + margin（需要定义绝对的margin）
    • 设置浮动后，另一个元素移动前一个元素的距离
  2. float + overflow
    • 开启了BFC
  3. display:table

# 三列布局（定宽+定宽+自适应）

  1. float + margin（需要定义绝对的margin）
    • 设置浮动后，另一个元素移动前一个元素的距离
  2. float + overflow
    • 开启了BFC
  3. display:table
